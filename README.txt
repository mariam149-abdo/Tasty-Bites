==============================================
 Restaurant Website 
==============================================

Project Name:
Tasty Bites - Restaurant Page

Author:
Mariam AbdlelhaK
--------------------------------------------------
 Description:
--------------------------------------------------
This is a clean and responsive restaurant menu webpage that displays pasta dishes with names, detailed descriptions, and discounted prices.

It's ideal for restaurant websites, food portfolios, or front-end development practice.

--------------------------------------------------
Features:
--------------------------------------------------
- Clean and modern layout
- Fully responsive design
- Shows old price and discounted price
- Easy to edit and reuse

--------------------------------------------------
Technologies Used:
--------------------------------------------------
- HTML5
- CSS3
-JAVASCRIPT
--------------------------------------------------
How to Use:
--------------------------------------------------
1. Download and extract the ZIP file.
2. Open `index.html` in any browser.
3. Customize the text, colors, or images if needed by editing `index.html` and `style.css` using a code editor (e.g. Visual Studio Code).
